Instructions

How to install eMARVault On an additional Computers

DESKTOP INSTALL INSTRUCTIONS
  1. Login into the website emarvault.com (Or eMAR Vault - Admin Panel (mytechwebsite.com))
  2. Select "System Settings" -> "Desktop Client"
  3. Locate version with "Latest" as the flag name. Right click on Download and copy link
  4. Login into the computer where you will be installing the eMARVault desktop client
  5. Download the client to the computer https://emarvault.com/download/7
  6. Run Installer from an admin account
  7. When prompted enter admin username (DTC or something like that) and the admin password
    1. This will create 2x scheduled tasks (eMARVaultHourlyCheck, eMARVaultHeartbeat)
    2. This will install the program into C:\Program Files (x86)\eMARVault

WEBSITE CONFIGURATION

  1. Login into the website emarvault.com (Or eMAR Vault - Admin Panel (mytechwebsite.com))
  2. Select "Computers"
  3. Locate the new computer that the software was installed on (OR sort by company name and the new device will show up first since there is no name there
  4. Click Edit (Pencil button next to the trashbin looking icon)
  5. Fill out the following fields
    1. Company Name (Drop Down if missing, setup company under the "Company Tab")
    2. Location (Drop Down if missing, setup "Location" under the "Location Tab" ) It will filter as you type in the company name
    3. Type (Enter description on where that computer is)
    4. SFTP Host (ftpus.pointclickcare.com)
    5. SFTP Username (emar backup username for that company)
    6. SFTP FolderPath (EG: emar_backup_hvhc_19)
    7. Scroll to the bottom and click "Save"

    Testing and Confirmation that its working
  6. Login into the computer and run the scheduled task
  7. OR = Wait up to 1 hour (Scheduled task runs 1x times per hour)
  8. A new ICON will show up on the desktop named "eMARVault"
  9. Click on it and it will prompt for a password to open the files
  10. The default password is = password
  11. This can be changed from the website by editing the computer and changing the "Folder Password" to something else.
